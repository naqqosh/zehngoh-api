# Zehngoh API

Backend service for Zehngoh local commerce platform.

## Tech Stack

- NestJS
- TypeScript
- PostgreSQL
- Prisma ORM

## Features

- Authentication
- User Management
- Product Catalog
- Categories
- Shopping Cart
- Order Management
- Seller Management
- Notifications

## Architecture

```markdown
src/
├── modules
├── common
├── prisma
├── config
└── main.ts
```

## Database

PostgreSQL + Prisma

## Deployment

Dockerized and deployed on DigitalOcean using GitHub Actions.
